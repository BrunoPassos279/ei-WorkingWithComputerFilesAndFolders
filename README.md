📂 Manipulação de Arquivos com Pathlib (Python)

Exemplos práticos usando pathlib para trabalhar com arquivos e diretórios em Python, do básico ao uso mais real.

🧠 O que é?

pathlib é a forma moderna (orientada a objetos) de manipular caminhos no Python — mais simples, legível e segura que os.path.
